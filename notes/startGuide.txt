go to docker-compose folder and execute below commands
    docker ps
    docker-compose -f common.yml -f kafka_cluster.yml up
    kafkacat -L -b localhost:19092

then run twitter-to-kafka-service

run elastic search with docker
    docker-compose -f common.yml -f elastic_cluster.yml up

before running elastic-query-service make sure to run config-server

for run elastic-query-web-client we only need config server and elastic-query-web-client after starting app
navigate to localhost:8184/elastic-query-web-client/home

run keycloak authorization server
    docker-compose -f keycloak_authorization_server.yml up
